# zk-categories.md — 分类码映射表

## 格式

每条记录：`{分类码}` | `{分类名}` | `{说明}`

## 映射表

| 分类码 | 分类名 | 说明 |
|--------|--------|------|
| aa | 法规框架 | 法律法规、监管规定 |
| bb | 操作实务 | 实操流程、步骤指南 |
| cc | 经济数据 | 宏观经济指标与分析 |
| dd | 地缘政治 | 地缘政治事件与影响 |
| ee | 货币政策 | 央行政策、利率、流动性 |
| ff | 资产配置 | 投资策略、大类资产 |
| gg | 合规风控 | 合规审查、风险控制 |

## 组序号规则

- `X0`（零后缀，如 aa0/bb0）：人工精化的参考级节点
- `X`（无零后缀，如 aa/bb）：自动蒸馏节点

## 知识节点命名

```
zk-{domain}-{catseq}-{docseq}-{title}.md
```

示例：`zk-ei-aa0-0-state-owned-equity-incentive-regulation.md`
