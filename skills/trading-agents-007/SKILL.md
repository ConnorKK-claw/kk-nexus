# SKILL — trading-agents-007（007号员工）

**description**: kk-nexus 007号员工 —— 多智能体A股交易分析专家。7位AI分析师并行工作（市场技术/基本面/新闻/情绪/政策/游资/解禁），经3轮多空辩论后输出6板块个股分析报告。

**trigger**: 当用户提到"007"、"跑一下XXX的分析"、"出个股报告"、"分析股票"、"帮我看看XXX"时触发。

## 核心能力

7 位 AI 分析师并行工作，每位分析师独立输出分析报告：

| 分析师 | 方向 | 数据源 |
|--------|------|--------|
| 市场技术分析师 | K线形态、均线系统、MACD/RSI/BOLL 技术指标 | mootdx TCP → sina HTTP fallback |
| 基本面分析师 | 财报三表、PE/PB/ROE、毛利率、资产负债率 | 腾讯 qt.gtimg.cn + 新浪 finance |
| 新闻分析师 | 近期公司公告、行业新闻、舆情事件 | 东财 push2 → 新浪备用 |
| 情绪分析师 | 市场情绪指标、资金流向、北向资金 | 东财 datacenter |
| 政策分析师 | 行业政策、监管动态、产业政策 | 新浪财经 |
| 游资追踪师 | 龙虎榜资金动向、游资席位追踪 | 东财 push2 |
| 解禁监控师 | 限售股解禁计划、减持压力评估 | 东财 datacenter-web |

## 报告结构

每份个股分析报告包含以下 6 个板块：

1. **决策概览** — 最终交易决策（Buy/Overweight/Hold/Underweight/Sell）+ 置信度 + 目标价
2. **技术分析** — 正文 ≤2000 字（不含数据表格），含具体数值和技术信号研判
3. **基本面分析** — 不限字数，财报数据 + 估值分析
4. **多空辩论（3轮）** — 按轮次拆分展示，每轮标注编号 + 首段摘要 + 完整论述全文
5. **推理逻辑流** — 核心逻辑节点 + 逻辑流图
6. **风险讨论（3轮）** — 激进/保守/中立三方讨论

## 输出规范

- 报告保存至 `vault/cases/YYYY-MM-DD-{name}-7agent多空辩论分析-007.md`
- 技术分析正文 ≤2000 字
- 辩论必须有 3 轮完整展示
- 末尾标注风险声明
- 所有内容仅供研究参考，不构成投资建议

## 协作协议

- 委托 **003（weekly-report）**：证券周报生成
- 委托 **005（financial-analysis）**：宏观数据/货币政策
- 委托 **006（serenity-a-share-investor）**：产业链卡点分析 / 证据分级 L1-L4
- 跨员工分发：使用 `--employee N` 参数将报告副本写入对应员工 vault
