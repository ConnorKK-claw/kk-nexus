# 企业所得税计算规则速查

## 税率速查

| 企业类型 | 税率 | 条件 |
|---------|------|------|
| 一般企业 | 25% | — |
| 小微企业 | 实际税负5%（减按25%计入应纳税额×20%税率） | 年应纳税所得额≤300万、人数≤300、资产≤5000万 |
| 高新技术企业 | 15% | 需认定 |
| 西部大开发企业 | 15% | 鼓励类产业 |

## 计算公式

应纳税额 = 应纳税所得额 × 适用税率 - 减免税额 - 抵免税额

## 预缴方式

- 按月或按季预缴
- 年终汇算清缴（次年5月31日前）

## 常见扣除标准

| 项目 | 扣除标准 |
|------|---------|
| 业务招待费 | 发生额60%与收入5‰孰低 |
| 广告费/业务宣传费 | 一般企业收入15%；化妆品/医药/饮料30% |
| 公益性捐赠 | 年度利润12%以内（超额可结转3年） |
| 研发费用加计扣除 | 未形成无形资产200%；形成无形资产按200%摊销 |
| 职工教育经费 | 工资薪金8% |
| 职工福利费 | 工资薪金14% |
| 工会经费 | 工资薪金2% |
