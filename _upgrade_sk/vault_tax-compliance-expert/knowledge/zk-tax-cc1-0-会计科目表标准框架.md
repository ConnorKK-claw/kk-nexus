# 会计科目表标准框架（企业通用）

## 资产类（1xxx）

| 编码 | 科目名称 | 说明 |
|------|---------|------|
| 1001 | 库存现金 | 企业库存现金 |
| 1002 | 银行存款 | 存入银行的各类存款 |
| 1012 | 其他货币资金 | 银行汇票/本票/信用卡存款等 |
| 1101 | 交易性金融资产 | 短期持有以交易为目的的金融资产 |
| 1121 | 应收票据 | 商业汇票 |
| 1122 | 应收账款 | 销售商品/劳务应收款项 |
| 1123 | 预付账款 | 预付的购货款/工程款 |
| 1221 | 其他应收款 | 除应收票据/账款外的其他应收暂付款 |
| 1231 | 坏账准备 | 应收账款备抵科目 |
| 1403 | 原材料 | 库存的各种材料 |
| 1405 | 库存商品 | 库存的外购商品/自制商品等 |
| 1471 | 存货跌价准备 | 存货成本高于可变现净值的差额 |
| 1601 | 固定资产 | 使用年限超1年的有形资产 |
| 1602 | 累计折旧 | 固定资产磨损价值的累计 |
| 1604 | 在建工程 | 在建的工程项目 |
| 1701 | 无形资产 | 专利权/商标权/著作权等 |
| 1702 | 累计摊销 | 无形资产摊销的累计 |
| 1801 | 长期待摊费用 | 已支付但摊销期超1年的费用 |
| 1901 | 递延所得税资产 | 可抵扣暂时性差异产生的所得税资产 |

## 负债类（2xxx）

| 编码 | 科目名称 | 说明 |
|------|---------|------|
| 2001 | 短期借款 | 1年内（含1年）的借款 |
| 2201 | 应付票据 | 商业汇票 |
| 2202 | 应付账款 | 采购应付未付款项 |
| 2211 | 应付职工薪酬 | 应付工资/社保/公积金/福利等 |
| 2221 | 应交税费 | 增值税/所得税/附加税等应交税费 |
| 2241 | 其他应付款 | 除应付票据/账款外的应付暂收款项 |
| 2501 | 长期借款 | 超过1年的借款 |
| 2502 | 应付债券 | 发行债券筹集的资金 |
| 2901 | 递延所得税负债 | 应纳税暂时性差异产生的所得税负债 |

## 权益类（4xxx）

| 编码 | 科目名称 | 说明 |
|------|---------|------|
| 4001 | 实收资本 | 投资者实际投入的资本 |
| 4002 | 资本公积 | 资本溢价/其他资本公积 |
| 4101 | 盈余公积 | 从净利润中提取的公积金 |
| 4103 | 本年利润 | 当年实现的净利润/亏损 |
| 4104 | 利润分配 | 利润分配各明细（未分配利润等） |
| 4201 | 库存股 | 回购本公司股票 |

## 成本类（5xxx）

| 编码 | 科目名称 | 说明 |
|------|---------|------|
| 5001 | 生产成本 | 产品生产直接成本 |
| 5101 | 制造费用 | 生产部门间接费用 |
| 5301 | 研发支出 | 研发活动支出 |

## 损益类（6xxx）

| 编码 | 科目名称 | 说明 |
|------|---------|------|
| 6001 | 主营业务收入 | 主要经营业务收入 |
| 6051 | 其他业务收入 | 非主要经营业务收入 |
| 6111 | 投资收益 | 对外投资取得的收益/损失 |
| 6301 | 营业外收入 | 非日常经营活动产生的利得 |
| 6401 | 主营业务成本 | 主要经营业务成本 |
| 6403 | 税金及附加 | 消费税/城建税/教育费附加等 |
| 6601 | 销售费用 | 销售环节各项费用 |
| 6602 | 管理费用 | 管理部门各项费用 |
| 6603 | 财务费用 | 筹资/利息/银行手续费等 |
| 6701 | 资产减值损失 | 资产减值形成的损失 |
| 6711 | 营业外支出 | 非日常经营活动产生的损失 |
| 6801 | 所得税费用 | 当期所得税+递延所得税 |
