# 股权激励对象变动处理规则

## 变动场景分类处理

### 1. 主动离职
- 已获授但尚未解锁的限制性股票：由公司按**授予价与市场价孰低**回购
- 已解锁部分：归激励对象所有，不做处理

### 2. 被动离职（公司解聘）
- 处理方式同主动离职
- 特殊说明：如涉及违规/渎职，公司可追回已解锁收益

### 3. 退休
- 按退休前在**考核期内（2024-2026年）的服务时间**折算首次授予权益
- 折算后的权益保留，待达到时间限制和业绩条件时继续解锁
- 未折算部分：按孰低原则回购

### 4. 职务变动——成为监事/独立董事
- 监事、独立董事按规定**不能成为激励对象**
- 已获授但尚未解锁的限制性股票：按**授予价格 + 银行同期存款利息**回购
- 利息不高于按回购时央行公布的同期限存款基准利率计算

### 5. 组织任命——职务晋升/调动
- 仍在公司体系内任职：**继续保留**已获授权益
- 调出公司体系：按退休处理（按服务时间折算）

### 6. 死亡/伤残
- 由**继承人或激励对象本人**继续持有已获授限制性股票
- 待达到时间限制和业绩条件时解除限售
- 也可协商由公司按孰低原则回购

### 7. 违规/渎职
- **追回**被回购之前已解除限售获得的全部或部分收益
- 未解锁部分：授予价与市场价孰低回购
- 公司有权视情节严重程度决定追回比例

## 规则来源

上述规则主要依据：
- 上海机场A股限制性股票激励计划（草案）第28-30页（变动处理章节）
- 国资发分配〔2006〕175号（基本框架）
- 国资考分〔2020〕178号（操作指引，细化规则）

## 实操提示

- **提前约定**：激励计划草案中应详细列明各类变动情形处理方式
- **书面确认**：激励对象变动时签署书面确认文件，避免争议
- **及时回购**：离职后应及时启动回购程序，避免股权结构长期不确定
- **税务衔接**：回购价款与激励对象原出资额的差额按工资薪金所得处理，详见bb0-2
