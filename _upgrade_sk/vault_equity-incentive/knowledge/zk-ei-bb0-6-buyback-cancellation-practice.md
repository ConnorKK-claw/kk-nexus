# 股权激励回购注销操作实务

## 触发条件清单

| 触发场景 | 适用阶段 | 回购价格 |
|---------|:-------:|---------|
| 业绩考核不达标 | 解锁前 | 授予价与市场价孰低 |
| 激励对象主动离职 | 限售期内 | 授予价与市场价孰低 |
| 激励对象退休 | 限售期内 | 按服务时间折算后孰低 |
| 激励对象成为监事/独董 | 限售期内 | 授予价 + 利息 |
| 激励对象违规/渎职 | 任何阶段 | 孰低 + 追回已解锁收益 |
| 公司终止激励计划 | 任何阶段 | 按实际情况 |
| 激励对象死亡/伤残 | 限售期内 | 继承人继续持有或孰低回购 |

## 回购价格计算规则

### 基本规则
- **孰低原则**：回购价格取授予价格与董事会审议回购事项前1个交易日公司股票交易均价的较低者
- **例外1（职务变动为监事/独董）**：按授予价格回购 + 银行同期存款利息
- **例外2（死亡/伤残）**：继承人可选择继续持有至解锁，或协商回购

### 派息调整
- 授予后如有派息，授予价格需相应调整
- 调整公式：调整后授予价 = 调整前授予价 - 每股派息金额

### 利息计算
- 利率：回购时中国人民银行公布的**同期存款基准利率**
- 利息不高于上述利率计算的金额

## 回购注销流程

1. **董事会决议**：审议回购注销事项，确定回购价格和数量
2. **公告披露**：说明回购原因、价格、数量、资金来源
3. **中登注销**：向中国结算上海分公司申请办理股份注销
4. **减资公告**：减少注册资本，通知债权人（刊登报纸公告）
5. **工商变更**：完成工商变更登记

## 案例：上海机场两次回购注销

### 第一次（2025-08-30）
- 原因：激励对象离职
- 公告文件：回购注销部分限制性股票及调整回购价格公告
- 涉及：同时调整回购价格（因派息调整授予价）

### 第二次（2025-10-31）
- 原因：部分A股限制性股票回购注销实施
- 状态：完成注销，发布实施公告

## 会计处理

回购限制性股票时的账务处理：
- 借：库存股（回购价格）
- 贷：银行存款
- 同时冲减对应的资本公积-股本溢价
