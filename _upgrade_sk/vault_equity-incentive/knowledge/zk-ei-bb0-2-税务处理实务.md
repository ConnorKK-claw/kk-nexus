# 股权激励税务处理与股份支付会计实务

## 股份支付会计处理

### 基本概念
股份支付：企业为获取职工服务而授予权益工具或承担以权益工具为基础确定的负债的交易。

### 结算类型
- **权益结算**：授予限制性股票、股票期权（最常见）
- **现金结算**：股票增值权、虚拟股票（以现金支付）

### 公允价值确定
| 工具类型 | 公允价值确定方法 |
|---------|---------------|
| 第一类限制性股票 | 授予日股价-授予价格 |
| 第二类限制性股票 | BS模型或蒙特卡洛模拟 |
| 股票期权 | BS模型 |

### 授予日股价影响
- 授予日股价越高 → 公允价值越大 → 股份支付费用越高
- 授予日选择影响财务报表

### 会计处理流程
1. **授予日**：确定公允价值，尚未确认费用
2. **等待期/锁定期**：按服务期限分摊确认费用
   - 借：管理费用/销售费用等
   - 贷：资本公积-其他资本公积
3. **可行权日及之后**：不再调整已确认费用
4. **未满足条件**：已分摊费用可冲回

### 回购股份的会计处理
- 回购时：借：库存股 贷：银行存款
- 授予时：借：银行存款 贷：库存股、资本公积

## 个人所得税处理

### 纳税义务发生时间
| 工具类型 | 纳税时点 |
|---------|---------|
| 限制性股票（第一类）| 解锁日 |
| 限制性股票（第二类）| 归属日 |
| 股票期权 | 行权日 |
| 股票增值权 | 兑现日 |

### 计税方法
- 属于"工资、薪金所得"，适用3%-45%超额累进税率
- 应纳税所得额 =（行权/解锁日市价-授予价）× 股数
- **不并入当年综合所得**，单独适用综合所得税率表

### 递延纳税政策（财税〔2016〕101号）
- **非上市公司**：递延至转让股权时纳税，按20%财产转让所得
- **上市公司**：一般不适用递延纳税
- **高新技术企业**：技术人员可申请分期缴纳（不超过5年）

### 多期激励的计税
- 同一纳税年度内多期行权/解锁，需合并计算税率
- 各期所得合并后按超额累进税率计税

### 备案要求
- 公司须向主管税务机关报送《股权激励计划备案表》
- 激励对象行权/解锁后，公司须履行代扣代缴义务
