# 股权激励信息披露与合规程序

## 总体审议程序

### 一般上市公司
1. 董事会薪酬考核委员会拟订草案
2. 董事会审议（关联董事回避）
3. 独立董事/监事会发表意见
4. 内部公示激励对象名单（≥10天）
5. 内幕信息自查（公告前6个月）
6. 股东大会审议（2/3以上通过，关联股东回避）
7. 60日内授予并完成登记

### 国有控股上市公司（以浦东新区为例）
1. 申报主体向区国资委政策咨询
2. 区国资委初审，反馈意见
3. 上市公司董事会审议草案
4. 申报主体报区国资委正式审核
   → 10个工作日内出具审核意见书
5. 股东大会审议（国有股东代表按国资委意见表决）
6. 60日内授予并完成登记

### 授予后撤回/重新申报的情形
- 董事会召开前撤销或股东大会未通过
  → 决议公告后5工作日内报告国资委
  → 3个月内不得再次审议
- 终止原计划、实施新计划、变更重要事项
- 6个月内未召开股东大会

## 信息披露时间线

### 草案公告时
- 董事会决议、激励计划草案及摘要
- 独立董事意见、监事会意见
- 法律意见书
- 财务顾问报告（如适用）
- 激励对象名单

### 股东大会通知时
- 法律意见书、财务顾问报告（如有）

### 股东大会决议后
- 股东大会决议公告
- 经审议通过的激励计划

### 实施过程中
- 授予/注销权益公告
- 条件成就公告（归属/解锁/行权）
- 权益价格/数量调整公告（除权除息等）
- 每期解锁/行权结果公告

### 终止时
- 终止决议公告
- 律师事务所意见书

## 窗口期管理

### 不得授予/行权的期间
| 事项 | 禁止期限 |
|------|---------|
| 年报/半年报 | 公告前15日（推迟则自原预约日） |
| 季报/业绩预告/快报 | 公告前5日 |
| 重大事件 | 发生日至披露日 |

### 实操要点
- 第二类限制性股票、股票期权可在窗口期内授予（不属于买入行为）
- 窗口期可从股东大会后60日期限中扣除

## 内幕信息管理
- 填写内幕信息知情人档案
- 填写重大事项进程备忘录（相关人员签字）
- 草案公告前6个月内买卖股票情况自查
- 股东大会后披露自查报告

## 中介机构要求
- **律师事务所**：必聘，出具法律意见书
- **独立财务顾问**：自主定价时强制聘请
