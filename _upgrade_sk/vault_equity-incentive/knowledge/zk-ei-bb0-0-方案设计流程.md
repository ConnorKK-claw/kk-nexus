# 股权激励方案设计全流程

## 概述
股权激励方案设计拆解为**四大模块、九大要素**。

## 四大模块

### 1. 方案篇
聚焦激励计划的核心要素设计：
- 主体资格、激励对象、数量来源、定价、考核、时间安排、异动处理

### 2. 程序及信息披露篇
从方案制定到终止的全流程操作：
- 方案制定 → 授予登记 → 解除限售/归属/行权 → 回购注销 → 方案变更调整终止

### 3. 会计及税务篇
- 股份支付会计处理原则
- 不同工具的费用计量与分摊
- 激励对象个人所得税计算、缴纳与递延政策

### 4. 特别篇—国有上市公司
- 国有控股上市公司特殊监管规定及审批程序

## 九大设计要素

### 1. 主体资格
- 非ST、*ST公司
- 近三年无重大违法违规
- 健全的公司治理结构

### 2. 激励对象
- 董事、高管、核心技术/业务骨干
- 不得包括独立董事、监事
- 单独持股5%以上股东需特别说明

### 3. 激励数量
- 全部有效计划涉及股票累计≤总股本10%
- 单个对象累计≤总股本1%
- 预留权益≤拟授予数量的20%

### 4. 股份来源
- 定向增发、二级市场回购
- 国资背景下近年倾向二级市场回购

### 5. 资金来源
- 激励对象自筹资金
- 公司不得提供贷款或担保

### 6. 激励价格
- 限制性股票：市价的50%左右
- 股票期权：行权价不低于市价
- 自主定价需财务顾问意见

### 7. 绩效考核
- 公司层面指标 + 个人层面考核
- 国资产控："3+N"指标框架

### 8. 时间安排
- 限售期/等待期：≥12个月
- 解锁/行权：分多期（常用3期，40%:30%:30%）
- 有效期：≤10年

### 9. 异动处理
- 激励对象离职/退休/死亡的处理
- 公司控制权变更/退市等特殊情形
