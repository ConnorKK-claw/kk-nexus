# 股权激励法规体系与时效性总览

## 法规层级结构

`
法律层 -> 《公司法》《证券法》《企业国有资产法》
证监会层 -> 《上市公司股权激励管理办法》（227号令2025年修正）
国资委层 -> 175号文 -> 171号文 -> 178号文 -> 102号文
地方国资层 -> 各省市实施细则（如上海303号文、浦东办事指南）
财税层 -> 35号文/5号文/461号文 -> 101号文 -> 2023年2号/25号公告
上交所层 ->《股票上市规则》2026年4月修订
`

## 实操建议

### 国有控股上市公司方案设计时应依循的法规优先级

1. **证监会227号令（2025年修正）** — 所有上市公司必须遵守的基础规则
2. **上市公司治理准则〔2025〕18号** — 2026年起适用，薪酬委审核替代监事会
3. **国资考分〔2020〕178号文** — 当前最完整的国资股权激励实操指引（地方参照执行）
4. **国资发考分规〔2019〕102号文** — 激励力度和范围的补充规定
5. **国资发分配〔2006〕175号 + 〔2008〕171号** — 作为补充，与178号文不冲突的部分继续有效
6. **地方国资规定** — 上海企业需同时遵循303号文和浦东办事指南

### 张江高科特别提示

1. **三重法规叠加**（证监会227号令 + 国资委175/171号 + 上海303号）——这是张江高科区别于非国企和央企的最大特征
2. **2022年审批权下放**——张江高科审批路径为张江集团->浦东新区国资委（无需市国资委）
3. **2025治理新规已生效**——薪酬与考核委员会替代监事会/独立董事审核
4. **单独计税延至2027年底**——须在此期限内完成首批解锁以享受此优惠
5. **国资委文件编号确认**：175号->171号->178号（发布顺序），178号央企为主地方参照
