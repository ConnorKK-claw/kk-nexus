# 激励工具选择

> 上市公司股权激励的主要工具包括限制性股票和股票期权。

## 简介

- **限制性股票**：以折扣价获得股票，锁定后分批解锁
- **股票期权**：未来以约定价购买股票的权利
- **第二类限制性股票**：无需提前出资，归属后缴款
