# 国有控股上市公司实施股权激励的六项条件

> 国有控股上市公司须同时满足以下条件方可实施股权激励：

1. **公司治理规范**：股东大会、董事会、监事会、经理层依法合规运作
2. **产权清晰**：不存在重大产权纠纷
3. **战略明确**：发展战略和经营计划清晰
4. **内控健全**：财务报告和内部控制制度完善
5. **业绩基础**：近3年无重大违法违规行为，业绩稳健
6. **劳动用工合规**：依法与激励对象签订劳动合同

## 负面清单

同《上市公司股权激励管理办法》第七条规定的不得实施的情形。
