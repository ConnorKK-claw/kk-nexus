# 2026-05-21 Memory Log

## 过程：全局检索系统升级

用户反馈：vault 检索和网页搜索时常出现信息错误
诊断：5 个根本原因（搜索引擎原始、无统一入口、无网页搜索、索引仅元数据、无降级链）

## 关键决策
- 工具装到 .codex/bin/ 全局路径，所有项目可用
- DuckDuckGo ddgs 作为网页搜索后端（免费、无 API Key）
- jieba + pypinyin + scikit-learn TF-IDF 为中文搜索基础
- 检索结果 < 2 条时自动降级到网页搜索

## 技术记录
- duckduckgo_search 已更名为 ddgs，pip install ddgs 才能用
- TF-IDF 需自定义 tokenizer 为 jieba.segment，否则中文分词不生效
- 大 vault（200+ 文件）搜索时，先快速子串预筛，再 jieba 验证，避免全文分词
- Windows PowerShell 下 Python 中文路径需注意转义
